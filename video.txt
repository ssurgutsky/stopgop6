================= ДРАКА 2 =========================

Кирпич бьет:

Удар правой -- уклон вниз, уход влево  | уход вправо
Удар левой  -- уклон вниз, уход вправо | уход влево
Удар правой -- уклон вниз, уход влево  | уход вправо
Удар левой  -- уклон вниз, уход вправо | уход влево

Передышка

Удар правой -- уход влево + удар левой | уход вправо
Удар левой  -- уход вправо + удар правой | уход влево

Передышка

Удар правой -- уход влево  | защита блоком, уход вправо
Удар левой  -- уход вправо | защита блоком, уход влево

Передышка

Удар левой -- уход вправо | защита блоком, уход влево
Удар левой -- уход вправо | защита блоком, уход влево

Передышка

В цикле, пока ГГ не выберет "уклон вниз + апперкот"
Удар правой -- уклон вниз + апперкот, уход влево  | защита блоком, уход вправо
Удар левой  -- уход вправо | защита блоком, уход влево

После апперкота Кирпич встает, но теряет силу

Кирпич idle -- ГГ удар правой ГГ удар левой // любой удар левой пробивает блок Кирпича
Кирпич idle -- ГГ удар правой ГГ удар левой


Удар правой -- уход влево + удар левой, защита блоком | уход вправо  // Кирпич не может пробить блок
Удар левой  -- уход вправо + удар правой, защита блоком | уход влево // Кирпич не может пробить блок

Передышка
Ушибленный Кирпич покачивается в стороны (fight2_gop2_fail_idle)

Кирпич idle -- ГГ удар правой, ГГ удар левой |   // удар правой пробивает блок Кирпича
Кирпич idle -- ГГ удар правой, ГГ удар левой |   // удар левой пробивает блок Кирпича
Кирпич idle -- ГГ удар правой, ГГ удар левой |   // удар правой пробивает блок Кирпича


Удар правой -- уход влево + удар левой, защита блоком | уход вправо   // (удар левой пробивает блок Кирпича)
Удар левой  -- уход вправо + удар правой, защита блоком | уход влево  // (удар правой пробивает блок Кирпича)

Передышка
Ушибленный Кирпич покачивается в стороны (fight2_gop2_fail_idle)

Кирпич idle -- ГГ удар правой, ГГ удар левой |   // удар правой бьет Кирпича по морде
Кирпич idle -- ГГ удар правой, ГГ удар левой |   // удар левой  бьет Кирпича по морде
Кирпич idle -- ГГ удар правой, ГГ удар левой |   // удар правой  бьет Кирпича по морде

Удар правой -- уход влево | уклон вниз, уход вправо + удар вправо // (уклон вниз - Кирпич пинает ГГ)
Удар левой  --  уход вправо + удар правой | уход влево + удар левой

Удар правой -- уклон вниз + апперкот // ФИНАЛ, Кирпич падает
